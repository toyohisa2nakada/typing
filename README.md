* タイピングゲームを作るための「ローマ字入力の正誤判定」を行うJavaScriptライブラリです。
* 例えば「ひっしゅう」は、hisshuu, hisshilyuu, hiltushilyuuなどと複数の入力パターンがありますが、それらを網羅した入力正誤判定を行うことができます。
* ひらがな と、ローマ字のどこまでをすでに打ったのかを得ることができます。
